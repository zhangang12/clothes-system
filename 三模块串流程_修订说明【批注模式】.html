v2.0 定稿
[基础资料] [部分员工] [财务] ├─ 工厂资料（S001 自动）─┐ │ │ └─ 客户资料（CN001 自动）─┤ │ │ ▼ 引用（实时同步） │ │ [样衣管理] │ │ 编号: S-20260528-01 │ │ 材料明细 / 寄样跟踪 / 三角色协作 │ │ │ │ │ ├─ 业务派单 → 版师做纸板 → 业务派 → 打样间做样衣 │ │ │ │ │ 【1:1】═════════════════════════════════════════════ │ │ │ │ │ ▼ 业务点 [📥 从样衣导入] │ │ [客户报价] │ │ 编号: Q-20260528-01 │ │ 报价明细（从样衣快照 + 损耗% + 含损金额） │ │ 费用明细（自动带 6 行：加工/线/包装/样衣/测试/运费） │ │ 报价合计（仅 3 字段：人民币价/美金价/备注） │ │ │ │ │ ▼ 主管审批 │ │ [已报价] → 发客户 │ │ │ │ │ ▼ 客户确认成单 │ │ [已成单] ══════════════════════════════════════════► 样衣列表"标绿" 提示 │ │ │ ▼ 转销售合同（全带，无审批） │ │ [销售合同] ──> 后续生产/出货/对账/付款流程 │ │ │ ▼ │ [财务付款]
📌 状态变化关键点：
  • 样衣管理状态独立（5 个：待派单/制版中/打样中/已寄出/已对账完成）
  • 客户报价状态独立（5 个：草稿/已报价/客户调整/已成单/已撤销）
  • 两个状态不直接联动，但报价"已成单" → 触发样衣列表标绿
  • 样衣可在任意状态下被业务建报价（B3 允许预报价）

⑦ 业务方 40 个问题 · 答案全清单

便于追溯

A. 数据流转：样衣 → 客户报价（6 问）

序问题业务方回答
A1样衣 → 报价 1:N 还是 1:1？1:1（要再报价必须复制新建）
A2颜色字段怎么映射？多颜色用 "/" 分隔合并到一个字段
A3实际耗用直接作为报价耗用？报价时业务可调整（如 ×1.05 加损耗）
A4供应商要不要带到报价？带，但 PDF 默认隐藏
A5样衣修改后报价同步吗？不同步（报价是快照）
A6样衣删除时关联报价怎么办？阻止删除（提示已有 N 张报价引用）

B. 状态机联动（3 问）

B1已成单是否反向更新样衣？列表中已成单的款号自动标绿，未成单不显示颜色（按颜色区分）
B2已撤销时样衣怎么处理？样衣状态不变（撤销只是这张报价的事）
B3样衣未完成时能提前出报价？允许（材料/耗用没填全也可估算填）

C. 引用一致性（3 问）

C1客户/工厂改了是否同步？实时同步（改名 → 所有引用单据自动更新）
C2客户资料删除怎么办？阻止删除（提示有 N 张关联单据）
C3工厂资料删除怎么办？阻止删除（提示有 N 张样衣单引用）

D. 编号体系（3 问）

D1两个模块编号会不会冲突？加前缀：样衣 S-20260528-01、报价 Q-20260528-01
D2客户/工厂编号自动还是手填？改自动生成（CN001 / S001 系统递增）
D3同一天序号怎么定？按当天创建顺序 1/2/3...

E. 货币 / 汇率（3 问）

E1板房样衣单价等什么币种？固定人民币 CNY
E2报价汇率从哪里来？业务填报价时手填（不从字典自动带）
E3汇率精度？保留 2 位小数

F. 角色权限（3 问）

F1版师/打样间能看报价吗？不能（报价是业务+客户之间的事）
F2业务在报价里改耗用？能改，且不影响样衣管理（独立快照）
F3财务能看样衣吗？能看，仅看对账金额相关字段

G. 附件 / 打印 / 导出（3 问）

G1附件是否带到客户报价？只带"样衣图稿"（让客户看款式）
G2PDF 报价单包含哪些？公司抬头/Logo、报价单号+询价日期、报价明细表、费用明细表、合计、样衣图片、业务员+联系方式、报价有效期（不含客户信息）
G3打印 vs 导出区别？打印=浏览器打印 A4；导出=生成 PDF 文件下载

H. 审批流（3 问）

H1样衣管理需审批？不需要，业务员直接创建保存
H2客户报价需审批？需要业务主管审批才能从「草稿」变「已报价」
H3报价改报价单需审批？不需要，业务员可直接修改（不做版本控制）

I. 报价单维护与分析（2 问）

I1报价单是否按客户分类统计？是，按客户成单率做简单统计
I2报价单合并发给客户？业务自由勾选合并（多个报价 PDF 合一）

J. 历史数据迁移（3 问）

J1历史样衣迁过来？不迁，新系统从零开始
J2历史客户报价迁过来？不迁
J3历史基础资料迁过来？不迁，重新录

K. 其他细节（5 问）

K1客户报价复制时怎么处理？复制新建，新建一张草稿；编号不复制（自动新生成）
K2客户停用后历史单据怎么处理？永久保留，不删数据；客户列表标记停用
K3样衣/客户报价 是否都需复制功能？都需要复制功能
K4样衣/报价的客户款号 可不同？可以不同（不强制等于样衣的款号）
K5客户/工厂改名后历史单据怎么处理？不处理（草稿/已报价 实时同步；已成单 保留旧名 — C1）
