📱 供应商门户 · 串流程待确认问题清单v1.0

⑥供应商协同门户(05 设计稿)是「材料合同 / 加工合同推送后,外部供应商 / 加工厂自助处理」的对外端:登录看自己的合同,按 🔖盖章 → 📦发货 → 🧾对账 → 💵开票 顺序操作,向上承接 ④订单 / ⑤合同,向下接 ⑦业务审批 / 财务付款。本清单梳理门户接入这条端到端业务流时必须由业务方拍板的衔接灰色地带,共 18 题,分 5 段。
怎么用:每题点选一个选项(标 推荐 的是我方建议),不满意可选「其他」并填写;每题下方「📍 出处」可点开对应设计稿分组核对;任意位置可 💬批注、可 ➕补充需求;填完点右上角 💾 保存 将答复写回本 HTML 文件回传即可。

💡 操作提示　① 直接在每题点选答案(单选),或选「其他」补充;② 每题下方「📍 出处」链接新标签打开 05 门户设计稿对应分组,可对照核对;③ 顶栏「💬 批注」开启后可对任意题目/段落写批注;④「➕ 补充需求」记录清单未覆盖的新需求;⑤ 改动后顶栏保存点变 橙点,点「💾 保存」写回文件。
