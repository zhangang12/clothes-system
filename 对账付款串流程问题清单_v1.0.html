💵 对账 · 开票 · 付款 · 串流程待确认问题清单v1.0

⑦对账-开票-付款(06 设计稿)是整条主线的终点:⑥供应商门户「我要对账」推来对账单 → 业务审批 → 供应商开票 → 系统自动生成付款申请 → 财务按账期付款。它向上承接 ⑤材料/加工合同(04)的合同单价与结算条款、⑥门户(05)的发货与发票。本清单梳理对账付款接入这条端到端业务流时仍需业务方拍板的灰色地带与最新反馈(款号检索 / 一单多合同 / 批次批注退回 / 分批付款 / 水单粘贴 / 无合同付款等),共 21 题,分 5 段。
怎么用:每题点选一个选项(标 推荐 的是我方建议),不满意可选「其他」并填写;每题下方 📍出处 链接可新标签打开对应设计稿定位;任意位置可 💬批注、可 ➕补充需求;填完点右上角 💾 保存 将答复写回本 HTML 文件回传即可。

💡 操作提示　① 直接在每题点选答案(单选),或选「其他」补充;② 顶栏「💬 批注」开启后可对任意题目/段落写批注;③「➕ 补充需求」记录清单未覆盖的新需求;④ 改动后顶栏保存点变 橙点,点「💾 保存」写回文件。
