📄 合同 · 业务规则
  • 🔑 两类合同:材料合同(原料/辅料购销协议,甲方=供应商/乙方=本司/丙方=担保可空)、生产加工合同(委托加工合同,受托方=加工厂/委托方=本司)。
  • 📋 编号:HT-YYYYMMDD-序号 + 类型标识;来源于④订单的【生成材料合同 / 生成加工合同】。
  • 🧩 一张合同可多选款号(D2);材料合同每种料的数量取订单算出的采购量,按整单/单料设定的分色/分码出明细行(参照拉链合同按颜色分行)。
  • 💰 加工合同价格包含项勾选:工缴/裁剪/线/后道/运费/纸箱/拷贝纸/干燥剂/运杂费/进仓费 + 增值税% + 不含项(胶袋) + 其他(手填) —— 取自真实加工合同写法。
  • 📑 合同基础条款(质量标准/交货/验收/结算账期/违约/争议/份数)为模板,大部分固定、可配置编辑(业务:基础模板见附件)。
  • 🖋 落款电子章:一键带入供应商上传的电子章,生成 PDF 时自动贴在落款盖章处(业务诉求);也可上传纸质盖章合同照片。
  • 📦 多次发货:在⑥门户「我要发货」中每次增加一条,业务审批(业务答复)。
  • 📋 下游单据编号(E4 定稿):对账单 DZ- / 发货单 FH- / 付款申请 FK-(本期先定义,与 HT- 同构)。
  • 🔒 快照/锁定:订单事后修改不反向同步已生成合同;合同被发货/对账引用后禁止删除,且锁定甲乙方/款号/数量/单价/金额(仅可改备注类字段,E5 定稿)。
  • 🔑 推送前需主管审批(D8 定稿),金额超阈值必审;推送=门户账号绑工厂资料、首次推送自动开通(B3)。
✅ 合同串流程问题清单 v1.0 · 已定稿（业务全部按推荐项确认 · 2026-06）
  • A 订单 → 合同:① 生成材料合同按供应商自动拆多张、生成加工合同按加工厂拆;② 关联款号仅同一订单内多选;③ 合同侧数量可微调、不回写订单;④ 订单不拆分→按款号汇总一行,拆分→按颜色/尺码分行;⑤ 货物明细标注数量来源(材料=采购量含损耗 / 加工=大货数)、可悬浮看公式;⑦ 交期默认偏移(加工−10天 / 材料−45天)可在系统参数配置、逐单可改;⑧ 已生成合同的订单禁删,关键字段改→给合同打「源订单已变更」标记;⑨ 生成合同后订单自动→「已生成合同」并显示关联合同号。
  • B 基础资料 → 合同:① 甲方(供方)必须先在工厂资料建档;② 工厂类型拆「材料供应商 / 加工厂」两类(一类一属)、合同按类型过滤;③ 推送=门户账号绑工厂资料、首次推送自动开通;④ 乙方/委托方取「本司主体」字典(支持多抬头);⑤ 代表默认登录业务员、可改选;⑥ 担保人现填+身份证照片加密存储、限授权可见;⑦ 工厂信息改→已生成合同快照不变。
  • C 样衣/报价 → 合同:① 材料单价默认带上游(报价/样衣)参考价、可改;② 拆单前强制校验供应商非空,空归「待定供应商」合同提示补全;③ 加工单价纯合同现填、可参考历史同款加工价;④ 材料照片默认复用样衣/订单照片、可替换;⑤ 列序统一订单口径(品名前、部位后);⑥ 物料以品名文本标识,用(品名+规格+颜色)做行内唯一键供发货/对账匹配(不建料库)。
  • D 合同内部:① 补料用「补料-原合同号」独立标识、不占 HT 主序号、挂原合同下;② 补料是独立合同、独立走门户、不回写订单用料核算;③ 条款模板按类型(材料/加工)各一套、可配置填空;④ 价格包含项仅汇入 PDF 文字不改金额,增值税 13% 注明含税不另计;⑤ 电子章系统维护、限授权可用,纸质盖章上传照片记录上传人/时间留痕;⑥ PDF 材料/加工各模板、多行分页、对供应商脱敏(仅显相关内容);⑦ 填担保人→ PDF 自动插入丙方担保条款;⑧ 合同推送前需主管审批、金额>阈值必审。
  • E 下游门户 / 对账:① 合同列表/编辑页显示门户进度(已推送→已盖章→已发货→已对账→已开票);② 对账=单价×已发货数量累计,合同数量为上限、超量预警、欠量多次补发;③ 材料/加工共用同一套门户与对账规则;④ 下游单据编号定义 DZ- / FH- / FK-;⑤ 已被发货/对账引用→锁定关键字段、仅改备注类;⑥ 接入可信时间戳/电子签章存证;⑦ 补料独立对账单、可按供应商汇总、付款随各自账期。
