📱 门户 · 业务规则
  • 🔑 对外端(首期 H5 网页,微信小程序后期·A1);账号由达泰后台为工厂开通并绑①工厂资料,凭专属链接/通知进入;供应商只见自己的合同与金额。
  • 🔐 登录(账号由公司开通,非自助注册):专属链接/通知仅深链定位到该合同、不免鉴权;首期 H5 主登录=手机号短信验证码 / 账号密码,凭与①工厂资料登记一致的手机号/账号进入(验证码仅校验持有权、不创建账号);微信仅作"微信内打开、已绑定账号的免登"增强(经公众号网页授权换 openid,需认证服务号 + 授权回调域名,会话过期需重登),不能凭微信号自助登录。
  • 🔒 四步顺序锁定:🔖盖章 → 📦发货 → 🧾对账 → 💵开票;前一步未完成后面置灰。
  • 🖋 盖章:一键电子章(自动贴 PDF 落款) / 上传纸质盖章照片。
  • 📦 发货:数量明细+物流+附件;多次发货每次增一条→业务审批。
  • 🧾 对账:合同单价×已审批发货数量自动算→勾选→推业务审批。
  • 💵 开票:上传 PDF 发票+配对账(发票=对账金额)→推财务。
  • 📨 推送供应商:内部在合同页点推送→生成专属链接/通知,供应商点开即进入对应合同。
  • 🏭 加工厂可见(A2·新增):加工合同推送后,加工厂在门户可查看该订单明细(材料清单 / 材料订单数 / 大货纸板 / 填充量 / 尺寸表 / 数量搭配)及供应商大货进度,便于配料与生产排期。
  • 👤 角色:内部=业务建合同/推送/审批对账;外部=供应商盖章/发货/对账/开票;财务=接收付款申请。
